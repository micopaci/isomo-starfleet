# Starfleet — Isomo EdTech Fleet Monitor

Fleet management platform for **40 Starlink sites** and **236 managed devices** across Rwanda.

## Stack

| Layer | Technology | Host |
|---|---|---|
| Backend API | Node.js 20, Express, PostgreSQL, WebSocket | GCP Cloud Run |
| Database | PostgreSQL 15 | GCP Cloud SQL |
| Web dashboard | Static HTML/CSS/JS (PWA) | Vercel |
| Desktop app | Electron 30, Vite, React 18 | Local build |
| Mobile app | React Native 0.74 (Android) | APK / managed Play |
| Shared client | TypeScript library | Workspace dependency |
| Windows agent | PowerShell 5.1 as SYSTEM | Microsoft Intune Remediations |

## Quick start

```bash
# Install dependencies
yarn install

# Start backend (runs migrations, then server)
npm run start --workspace=packages/backend

# Start backend in dev mode (auto-restart)
npm run dev --workspace=packages/backend

# Start desktop app
npm run dev --workspace=packages/desktop

# Start mobile Metro bundler
npm run start --workspace=packages/mobile
```

## Environment

Copy `.env.example` and fill required variables. Key groups:

| Group | Variables |
|---|---|
| Database | `DATABASE_URL` (Cloud SQL connection string) |
| Auth | `JWT_SECRET` or `JWT_PRIVATE_KEY` + `JWT_PUBLIC_KEY` |
| Microsoft Graph | `GRAPH_TENANT_ID`, `GRAPH_CLIENT_ID`, `GRAPH_CLIENT_SECRET` |
| Intune triggers | `REMEDIATION_POLICY_DIAGNOSTICS`, `REMEDIATION_POLICY_LOCATION_REFRESH`, etc. |
| Notifications | `SMTP_*`, `DIGEST_*`, `DASHBOARD_URL` |

## Monorepo layout

```
packages/
  backend/     ← Express API, migrations, scheduled services
  web/         ← Static dashboard (Vercel + same-origin backend)
  desktop/     ← Electron operator console
  mobile/      ← React Native Android app
  shared/      ← API client, WebSocket client, hooks, types
  agent/       ← Windows PowerShell telemetry agent
```

## Deployment

| Service | Path |
|---|---|
| Backend | Push to `main` → GitHub Actions → Cloud Run staging → promote to production |
| Database | Cloud Run migration Job runs `node migrate.js` before each deploy |
| Web | Push to Vercel-tracked branch, dashboard loads at `starfleet.icircles.rw` |
| Desktop | `npm run build --workspace=packages/desktop` |
| Mobile | React Native Android build from `packages/mobile` |
| Agent | Intune Remediation scripts generated by `packages/agent/build-intune-*.mjs` |
